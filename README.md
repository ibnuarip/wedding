# Wedding Invitation Website

Website undangan pernikahan digital yang modern, responsif, dan elegan. Dirancang untuk memberikan pengalaman yang lancar bagi para tamu undangan di berbagai perangkat.

## Fitur Utama

- **Desain Responsif**: Tampilan yang optimal di perangkat mobile, tablet, dan desktop.
- **Hero Section**: Latar belakang romantis dengan hitung mundur (*countdown timer*) yang dinamis.
- **Cerita Kami**: Timeline interaktif yang menceritakan perjalanan cinta pasangan.
- **Galeri Foto**: Menampilkan momen-momen indah dengan efek lightbox.
- **Konfirmasi Kehadiran (RSVP)**: Formulir RSVP yang dilengkapi dengan notifikasi toast modern.
- **Kolom Komentar**: Integrasi Disqus untuk tamu memberikan ucapan dan doa.
- **Musik Latar**: Kontrol musik yang estetik untuk mengiringi pengalaman tamu.

## Teknologi yang Digunakan

- **Struktur**: HTML5 (Semantik)
- **Styling**: CSS3 & Bootstrap 5
- **Logika**: JavaScript (Vanilla JS ES6+)
- **Font**: Google Fonts (Sacramento & Work Sans)
- **Ikon**: Bootstrap Icons

## Cara Menjalankan

1. Clone atau download repositori ini.
2. Buka file `index.html` menggunakan browser pilihan Anda.
3. Untuk pengalaman terbaik (seperti fitur komentar Disqus), jalankan menggunakan server lokal (seperti Live Server di VS Code).
